# Data Jobs Dashboard
![Dashboard Page](/images/Page_1.png)
## Introduction
This dashboard provides a comprehensive overview of the data jobs market, designed to help job seekers, professionals, recruiters and organizations stay informed on industry trends and opportunities.
## Skills showcased
* **Data collection, cleaning and transformation**: Extracting, transforming and preparing job market dataset for analysis.
* **Data visualization**: Creating clear interactive charts and dashboards to highlight trends.
* **KPI indicators and tables**: Used cards to display key metrics and **tables** to provide granular, sortable data.
* **Interactive reporting**: I achieved an interactive dashboard using the **slicers, buttons & bookmarks, and drill-through** features.
## Dashboard Overview
### Page 1: Main Dashboard
![Dashboard Page 1](/images/Page_1.png)
This page gives an overview of what is happening in the data jobs market at a glance. It showcases key KPIs like total job count, median salaries, top job titles and so on.
### Page 2: Job Title Drill Through
![Dashboard Page](/images/Page_2.png)
From the main dashboard, you can drill through to this view to get specific details for a single job title, including salary ranges, work-from-home stats, whether a degree is required, health insurance stats, job type, and top hiring platforms.
## Conclusion
This dashboard brings clarity to job posting data, which helps users make informed decisions about the career paths. Also, can serve as a powerful tool to employers for career analysis.
